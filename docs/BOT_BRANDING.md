# BOT BRANDING

## Рекомендованный основной вариант
- Название бота: `Белита Skin Match`
- Username для BotFather: `BelitaSkinMatchBot`
- Короткое позиционирование: AI-помощник по косметике Belita/Vitex с объяснением состава простым языком.

## Альтернативные username
- `BelitaCareBot`
- `BelitaInciBot`
- `BelitaSkinGuideBot`

Статус: это предложенные варианты; фактическую доступность username нужно проверить в `@BotFather`.

## Short Description
`Подбираю косметику Belita/Vitex по составу, типу кожи и вашему запросу. Объясняю, почему средство подходит. Создано @AI_Nikitka93.`

## About
`Белита Skin Match — Telegram-бот для подбора косметики Belita/Vitex по INCI-составу, типу кожи и целям ухода. Бот объясняет рекомендации простым языком, помогает найти более мягкие формулы и не заменяет врача. Создано @AI_Nikitka93.`

## Welcome Message
`Привет! Я Белита Skin Match. Помогу подобрать косметику Belita/Vitex по составу и вашему запросу. Я не врач, но могу аккуратно подсказать более подходящий уход и объяснить, почему именно эти средства выглядят уместно для вашей кожи.`

## Tone of Voice
- Спокойный
- Доброжелательный
- Объясняющий без медицинских обещаний
- Понятный даже без знания INCI

## Почему выбран этот вариант
- `Skin Match` звучит современно и понятно для Telegram-бота.
- В названии сохраняется сильная привязка к бренду `Belita`.
- Формулировка хорошо масштабируется на Mini App и Web без ребрендинга.
